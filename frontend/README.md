# NiyalGPT Frontend

React/Vite frontend for the NiyalGPT chatbot experience.

## Scripts

```bash
npm install
npm run dev
npm run build
npm run preview
```

## Environment

Copy `.env.example` to `.env` and set:

```env
VITE_API_URL=http://localhost:5000/api
```

For production, point `VITE_API_URL` to the deployed backend API.
